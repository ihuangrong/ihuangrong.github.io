每周德语语法：动词的格

发表于 2020-05-23 分类于 学习 ， 德语学习 Deutschlernen 阅读次数：

学习德语动词时首先要想一下动词后面跟几格。
第一格 Nominativ；第二格 Genitiv；第三格 Dativ；第四格 Akkusativ
主语是第一格。

z.B.
In Deutschland trinkt man gern Bier. 在德国人们喜欢喝啤酒。
In der Schweiz spricht man Deutsch. 在瑞士人们说德语。
man是不定代词，在句子中充当主语后面的动词变位是第三人称单数

一格动词：

动词后面跟第一格 Verben mit Nominativ

  1. sein
    z.B.
    Er ist ein guter Arzt.
    Das ist ein gutes Auto.
    Sie ist eine gute Lehrerin.
  • 表示计量、单位时，sein后面跟第四格
    z.B.
    Der Tisch ist einen Meter lang.
    Das Baby ist einen Monat/ein Jahr alt.
    Das Buch ist einen Euro teuer.
  • 固定结构中，sein后面跟第二格
    表达观点、看法：
    Ich bin der ( 二格 ) Meinung, dass … 我认为…
    Ich bin der Ansicht, dass… 我的看法是…
    Ich bin anderer Meinung / Ansicht. 我有不同看法。
    表达心情、情绪：
    Ich bin heute guter Laune. / Ich bin heute schlechter Laune.

  1. werden
    z.B.
    Er wird ein guter Lehrer. / Sie wird eine gute Lehrerin.
    Er möchte ein guter Lehrer werden.

  2. bleiben
    z.B. Er bleibt ein guter Lehrer. 他依然是一位好老师。

四格动词：

动词后面跟第四格 Verben mit Akkusativ
freuen jn.A 使某人感到高兴
z.B. Das Ergebnis freut mich. “ Ich freue mich über das Ergebnis. 我对这个结果感到高兴。

interessieren jn. A 使某人感兴趣
z.B. Das Buch interessiert uns alle.

enttäuschen jn. A 使某人感到失望
z.B. Das Ergebnis enttäuscht meinen Vater.

verschmutzen A 污染
z.B. Die Industrie verschmutzt die Luft. 工业污染了空气。

kaputtmachen A 弄坏
z.B. Ich habe den Fernseher kaputtgemacht.

zerstören A 摧毁
z.B. Das Hochwasser zerstört viele Städte. 洪水摧毁了许多城市。

  • 双四格动词：
    kosten 花费某人多少钱 ( 连续两个第四格 )
    z.B. Das Buch kostet ihn einen Euro. 这本书花了他一欧元。

  • 词组中固定第四格：
    eine Rede halten 发表讲话，Rede就是halten的第四格

三格动词

  1. 正向三格动词：一格的动作作用在三格
    helfen D
    z.B. Ich helfe einem alten Mann.

    danken D
    z.B. Sie dankt ihrem Mann.

    gratulieren jn.D zu etw.D 就某事祝贺某人
    z.B. Die Bundeskanzlerin Angela Merkel gratuliert dem US-Präsidenten zu seinem Wahlsieg.

    gehören D 属于某人
    z.B.
    Das Auto gehört mir.
    Wem gehört das Auto?

    passen D 适合某人，stehen D ( 颜色、外型 )适合某人
    z.B. Das Hemd passt mir gut. / Das Hemd steht mir gut.

    entsprechen D 与…符合，反义词：widersprechen D 与….不符合
    z.B. Das Ergebnis entspricht/widerspricht meinem Wunsch.

  2. 反向：三格的动作作用在一格
    gefallen D 喜欢，反义词：missfallen 不喜欢 物一人三
    z.B. Das Auto gefällt/missfällt mir.

    fehlen D 缺乏
    z.B. Geld fehlt mir. 我缺钱。

    mangeln D 缺乏

    schmecken D 味道好
    z.B. Die Suppe schmeckt dem Kind nicht. 这个孩子不喜欢喝这个汤。

双宾动词：

间接宾语 D；直接宾语 A
empfehlen 给某人推荐某人或某物
z.B.
Er empfiehlt meiner Mutter eine Restaurant.
Er empfiehlt meiner Mutter einen besseren Lehrer.

vorstellen 给某人介绍某人或某物
z.B.
Darf ich euch meine Frau vorstellen?
Ich will euch meinen Sohn vorstellen.
Ich will euch das historische Rathaus vorstellen.

kaufen 给某人买某物
z.B.
Mein Vater hat meiner Mutter eine Halskette gekauft.
Meine Mutter hat meinem Vater eine Halskette gekauft.

二格动词：

动词后面跟第二格 Verben mit Genitiv
gedenken G 纪念
z.B.
Wir gedenken der Opfer(Pl.). 我们怀念遇难者。
Wir gedenken des früheren US-Präsidenten.
